# OpenPegasus
Open Source Firmware for the Pegasus Touch 3D Printer

   Compile:

        cd mainapp
        scons
	cd hardware
	./compileLaser

   Running (from mainapp/)
      Non-GUI Run:

        bin/gui calib.txt <gcode file>

      GUI run (unstable):

        bin/gui


Generating GCode

   I used slic3r v.

   I use this configuration file: slic3r/pegasus.ini
